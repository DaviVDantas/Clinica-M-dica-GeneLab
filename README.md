## 🎓 Contexto Académico

Este projeto foi desenvolvido como requisito obrigatório (**Projeto 01**) para a atividade de **Sistematização** da disciplina de **Programação e Desenvolvimento Web** do **CEUB (Centro Universitário de Brasília)**.

**Objetivos da atividade cumpridos:**
- [x] Título da aba configurado.
- [x] Mínimo de 2 imagens dos produtos/serviços.
- [x] Descrição clara dos serviços oferecidos.
- [x] Secção de equipa fictícia (mínimo 3 membros com foto e cargo).
- [x] Hierarquia semântica com pelo menos 2 níveis de cabeçalhos (`H1`, `H2`, `H3`).
- [x] Formulário estático de pré-agendamento (Nome, E-mail, Cidade, Estado).

---

## ✨ Diferenciais Técnicos (UI/UX)

Além dos requisitos básicos, o projeto conta com:
* **Design Glassmorphism:** Painéis translúcidos e efeitos de vidro fosco para um visual futurista.
* **Animações 3D:** Fundo interativo gerado dinamicamente no Canvas utilizando a biblioteca **Three.js**.
* **Cursores Personalizados:** Interações magnéticas com botões e links.
* **Totalmente Responsivo:** Layout adaptado para dispositivos móveis, tablets e desktops (Mobile First).
